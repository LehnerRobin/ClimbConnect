# Setup-Anleitung (Entwurf)

## Voraussetzungen
- Git & GitHub
- .NET SDK (geplant: .NET 8)
- Node.js & Angular CLI
- Datenbank (z.B. Oracle-Testinstanz oder später Docker)

## Geplante Schritte
- Backend in `backend/` (ASP.NET Core Minimal API)
- Frontend in `frontend/` (Angular)
- Datenbankschema in `database/` (`schema.sql`, `data.sql`)

> Konkrete Befehle (dotnet, ng, etc.) werden ergänzt, sobald die Implementierung startet.
